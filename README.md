# yarn-nx-project

## Getting started

```
$ yarn install
```

## Commands

### App

#### Build / Generate

```
$ yarn nx run app:build
$ yarn nx run app:generate
```

#### Dev / Start

```
$ yarn nx run app:dev
$ yarn nx run app:start
```

### E2E

```
$ yarn nx run e2e:run
```

### Lint

Run lint script across all workspaces:

```
$ yarn nx run-many --target=lint --all
```
